A password safe written in Go implementing the http://pwsafe.org/[Password Safe] version 3 database format.

== How it works

gopwsafe runs entirely in your browser: a Go binary compiled to WebAssembly handles all cryptography and database parsing client-side, so your password database and master password never leave your device.
The Svelte frontend loads the WASM module, lets you open or create a `.psafe3` file from your local filesystem, and stores nothing remotely.
Because it is packaged as a Progressive Web App (PWA), it can be installed to your home screen and used fully offline.

== Installation

=== Install as a Web App (recommended)

Visit https://backgroundprocess.com/gopwsafe in a modern browser (Chrome, Edge, Firefox, or Safari).
Click the *Install* button that appears in your browser's address bar or the in-page install prompt to add gopwsafe to your home screen or app launcher.
Once installed it works offline without any further setup.

=== Self-host / compile from source

**Prerequisites:** Go 1.21+, Node.js 18+

1. Build the WebAssembly binary:
+
[source,bash]
----
GOOS=js GOARCH=wasm go build -o pwa/public/gopwsafe.wasm ./cmd/wasm/
----

2. Build and preview the frontend:
+
[source,bash]
----
cd pwa
npm install
npm run build
npm run preview
----

The app requires HTTPS (or localhost) for the Clipboard and File System Access APIs to work.

== Security notes

All cryptography runs client-side in WebAssembly — your master password and database contents never leave your device.

Browser extensions with host permissions can inject JavaScript into any page, including this one, and could read decrypted passwords from memory.
This is a limitation of any browser-based password manager.
For the strongest isolation, install gopwsafe as a PWA on *iOS or iPadOS* (Add to Home Screen) or *macOS Safari* (Add to Dock): home-screen web apps on these platforms run in a dedicated process where browser extensions do not execute.
